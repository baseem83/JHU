Considerations

  1. Outputs Population for each of the next five years
  2. Inputs
    • Current Population
    • Birth Rate
    • Death Rate
    • Immigration Rate
  3. Input workflows Program prompts user for each input
  4. Permissible editing
    • Population cannot be negative
    • Birth and death rate cannot be negative
  5. Information persistence This is calculator that displays information. No saving is necessary
  6. Assumptions
    • The program will always output a 5 year projection. This should be discussed with the project sponsor for confirmation. If it may change, this can be added to the list of inputs
    • The birth/death/immigration rates do not change annually, and are instead fixed
    • One year is 365.25 days. Confirm project sponsor agrees, or would they rather match to a calendar year, with the possibility of a leap year
    • All rates are being input as a ratio of second (num / sec)
  7. Problem simplification
    • Input and data cleansing
    • Calculation
    • Format and output
  8. Alternate solutions Vary input options (such as procuring info from a file)
